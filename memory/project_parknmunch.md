Full website built at `c:\Users\abdul\Website Building\ParkNMunch\`. Frontend-only demo (no backend/payments).

Structure: `index.html`, `order.html`, `css/style.css`, `css/order.css`, `js/menu-data.js`, `js/main.js`, `js/order.js`, `images/` (empty — drop images here).

**Why:** Client wanted a premium park-and-eat takeaway site where customers order from their car; car registration is a key UX requirement.

**How to apply:** Image files go in `images/` matching filenames in menu-data.js (e.g. `rs3.jpg`, `m5.jpg`). The site degrades gracefully with emoji placeholders when images are absent.
